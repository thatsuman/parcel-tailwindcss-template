# Parcel + Tailwind CSS Starter Template

Scaffold a new project with bundler [Parcel.js](https://parceljs.org/) and [Tailwind CSS](https://tailwindcss.com/) preconfigured with ease.

## Configured with

- Parcel.js
- TailwindCSS

## Installation

```sh
yarn
```

## Development

```sh
yarn parcel
```

## Build

```sh
yarn build
```
